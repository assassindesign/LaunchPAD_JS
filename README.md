# LaunchPAD_JS
LaunchPAD on JavaScript

Usage
	in SelectBox, choose Unipack name (ex: faded, dont let me down, k-pop culture)
 	then you can autoplay or click, button input
  
Button List
	a-z, number 1-9 0, spacebar, numpad 0-9 / * - + del, - = backspace [ ] \ ; ' , . / `
